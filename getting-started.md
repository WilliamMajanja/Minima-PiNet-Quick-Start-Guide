## What is Minima-PiNet-OS?

Minima-PiNet-OS is a Raspberry Pi-focused ARM64 operating environment designed around:

- **Zero-bloat runtime philosophy**
- **Web3-native infrastructure (Minima + IPFS)**
- **Integrated DApp platform**
- **AI acceleration support (CPU/NPU optimized paths)**
- **Enterprise-style security and attestation patterns**

## Who this wiki is for

- Makers building edge compute systems
- Developers shipping DApps on constrained devices
- Operators running self-hosted Web3 infrastructure
- Teams exploring trusted Raspberry Pi clusters

## Minimum prerequisites

- Raspberry Pi 4/5 (Pi 5 recommended for enterprise features)
- ARM64-compatible storage medium (microSD or NVMe setup)
- Stable network access for provisioning, package retrieval, and node sync
- Basic Linux CLI familiarity for operations and debugging
