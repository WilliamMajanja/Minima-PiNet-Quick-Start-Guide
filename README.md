## GitHub Pages hosting

To host this wiki as a static GitHub Page:

1. Open repository **Settings → Pages**.
2. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
3. Select branch **`copilot/create-wiki-for-repository`** (or your default branch after merge) and folder **`/(root)`**.
4. Save, then open the published site URL shown in Pages settings.

The site uses Jekyll with the default GitHub Pages-compatible `minima` theme via `_config.yml`.
