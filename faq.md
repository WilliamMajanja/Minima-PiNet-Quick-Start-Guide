## Is this wiki the upstream project?

No. This repository hosts documentation for the upstream project at [WilliamMajanja/Minima-PiNet-Os](https://github.com/WilliamMajanja/Minima-PiNet-Os).

## Can I use this on Raspberry Pi 4?

Yes, but newer enterprise and acceleration paths are optimized for newer hardware configurations.

## Do I need to use DApps?

No. You can run the platform as a minimal Web3/edge node and add DApps only when needed.

## What DApp types are supported?

TypeScript package-based DApps, URL-based React dashboards, and classic Minima MiniDapps.

## How should I secure production deployments?

Use least privilege, private networking, strict endpoint exposure, regular updates, and integrity verification workflows.

## Where do I report issues or feature requests?

Use the upstream repository issue tracker: <https://github.com/WilliamMajanja/Minima-PiNet-Os/issues>
